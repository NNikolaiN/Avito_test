## Найденные баги

### P0 — Критические:

- P0 — Фильтр по цене не работает корректно: при диапазоне от 1000 руб. отображаются объявления стоимостью 500 руб.
- P0 — Модерация не сработала: в выдаче присутствует объявление с некорректной фотографией (плюшевый персидский кот вместо живого персидского кота)
- P0 — Фильтр по цене не работает: при установленном диапазоне от 1000 до 30000 отображаются объявления с ценой 40000 и выше
### P1 — Высокие:

- P1 — Неправильно отображается количество объявлений: указано «Котята в Москве — 19», но фактически отображается только 8 объявлений
- P1 — Фильтр по породе работает некорректно: при выбранной породе отображаются объявления других пород
- P1 — Фильтр по полу работает некорректно: при выбранном значении «Мальчик» отображаются объявления с полом «Девочка»
- P1 — Сортировка объявлений работает некорректно: первым в выдаче отображается объявление из другого города
### P2 — Средние:

- P2 — Неправильный порядок хлебных крошек (breadcrumbs): «Животные → Персидская → Кошки» вместо «Животные → Кошки → Персидская»
- P2 — На странице отображается ложное сообщение об ошибке («Ошибка страницы»), хотя страница загружается нормально
### P3 — Низкие:

- P3 — Некорректный alt-текст у кнопки «Избранное» (вводит пользователя в заблуждение)
