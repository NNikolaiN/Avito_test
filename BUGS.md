## BUG-01: Фильтр «Диапазон цен» работает некорректно

**Priority:** P0 (Критический)  

**Окружение:** Desktop (Chromium)

**Шаги воспроизведения:**
1. Открыть главную страницу  
2. Ввести диапазон цен (например 0–1000)  
3. Дождаться обновления списка объявлений  
4. Проверить цены отображаемых объявлений  

**Фактический результат:**
- В выдаче присутствуют объявления, не попадающие в указанный диапазон  

**Ожидаемый результат:**
- Все объявления должны удовлетворять условию:  
  `min_price ≤ price ≤ max_price`  

**Дополнительно:**
- Баг воспроизводится для диапазонов:
  - 0–1000  
  - 1000–3000  
  - 3000–10000

## BUG-02: Тоггл «Только срочные» работает некорректно

**Priority:** P0 (Критический)  

**Окружение:** Desktop (Chromium)

**Шаги воспроизведения:**
1. Открыть главную страницу  
2. Включить тоггл «Только срочные»  
3. Проверить отображаемые объявления  

**Фактический результат:**
- В выдаче присутствуют объявления без признака срочности  

**Ожидаемый результат:**
- Все объявления должны быть помечены как срочные  
- Несрочные объявления не должны отображаться

## BUG-03: Кнопка запуска таймера не работает корректно

**Priority:** P1 (Высокий)  

**Окружение:** Desktop (Chromium)

**Шаги воспроизведения:**
1. Открыть страницу статистики  
2. Нажать кнопку паузы  
3. Нажать кнопку запуска таймера  

**Фактический результат:**
- Сообщение «Автообновление выключено» остаётся на странице  

**Ожидаемый результат:**
- Сообщение «Автообновление выключено» исчезает  
- Таймер возобновляет работу
